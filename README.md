# Система распознавания лиц для контроля перемещения сотрудников

## Описание проекта
Разработка ИИ-системы распознавания лиц сотрудников с целью контроля перемещения по объектам предприятия. В долгосрочной перспективе система может быть интегрирована в систему видеонаблюдения, что даст возможность оперативно восстановить последовательность событий в случае непредвиденных ситуаций.

## Мои задачи и вклад
- Создание и обработка датасетов для обучения моделей распознавания лиц
- Проведение многочисленных экспериментов с различными библиотеками и алгоритмами распознавания
- Разработка среды разработки для эффективной работы с инструментами компьютерного зрения
- Реализация и адаптация кода для работы с различными библиотеками распознавания лиц

## Использованные технологии
- **Библиотеки распознавания лиц:** OpenCV (Haar), MTCNN, ArcFace, DeepID, Facenet, DeepFace, Dlib, InsightFace (retinaface), Face Recognition, YuNet Face Detection
- **Системы трекинга:** YOLOv8, AutoTrackAnything, SORT, DeepSort, FairMOT, ByteTrack, BoT-SORT
- **Инфраструктура:** GStreamer, NVidia TensorRT, NVidia DeepStream
- **Хранение данных:** FAISS, PostgreSQL + pgvector
- **Фреймворки:** PyTorch

## Результаты
- Создан конвейер распознавания лиц в видео для различных сцен в офлайн-режиме
- Реализована комбинированная система трекинга лиц YOLOv8 + BoT-SORT + InsightFace
- Разработано REST API на базе FastAPI с настраиваемым конвейером обработки
- Создана векторизованная база данных детектированных лиц
- Оптимизирована скорость работы конвейера путем вычислений только в зоне интереса и уменьшения площади изображений для обнаружения лиц
